# PacsImgShow PACS图像管理与查看工具

## 功能介绍
- 图像管理：支持从PACS服务器检索图像，包括查询、检索、存储等操作。
- 图像查看：提供图像查看功能，支持缩放、旋转、拖动等操作。
- 图像标注：支持在图像上添加标注，包括矩形、圆形、箭头等。
- 图像导出：支持将图像导出为常用格式，如JPEG、PNG等。
- 支持在线标注：支持在图像上进行标注，标注信息可以实时同步到PACS服务器。
- 支持多用户登录：支持多个用户同时登录，每个用户可以独立操作。
- 支持Web端访问：支持通过浏览器访问PacsImgShow，无需安装客户端。

## 特性
- 跨平台：支持Windows、macOS、Linux等操作系统。
- 简单易用：提供简单易用的界面，无需专业知识即可操作。
- 使用VUE3框架开发，提供响应式以及现代化界面

## 技术栈
- 前端：Vue3 + ElementPlus
- 后端：Node.js + Express (暂时不做)
- 数据库：SQL Server数据库(暂时不做)
- 认证：JWT（JSON Web Token）(暂时不做)

## 项目构件

```bash
$ npm install
```
